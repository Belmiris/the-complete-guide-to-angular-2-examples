98. Injecting the Logging Service into Components

Remember to put the name of your service into a new 'providers' array in your component.

Angular Dependency Injection is Hierarchical.

This is counter-intuitive so note: A new Service is created when the name of the Service is in the providers array of the component / directive

providers: [myService]

A Service created in a module is the same instance for that module and all child modules.

Create a Service in the App Module if you want the entire application to see it.

Injecting Services into another service is tricky. You need to provide the parent Service with meta-data by decorating it with @Injectable.
